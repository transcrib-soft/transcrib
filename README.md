# Transcrib

🎙️ Electron-приложение для транскрибации аудио и видео в текст на основе **Whisper** (OpenAI) и **whisper.cpp**.

## Возможности

- Встроенные модели: **tiny** и **base**
- Дополнительные модели: **small**, **medium**, **large**
- Поддержка 99+ языков
- Экспорт в TXT, JSON, SRT, VTT
- Тайм-коды в результатах
- Управление лицензией

## Системные требования

- **OS:** Windows 10 / 11 (x64)
- **RAM:** от 4 ГБ (рекомендуется 8 ГБ)
- **Диск:** 2–5 ГБ для моделей
